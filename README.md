# Tech Stack
- ReactJS ⚛️
- Tailwind 🌊
- Recharts 📊

# Installation

1. Clone the repo

    ```sh
    git clone https://github.com/pranikz/Vacalysis-frontend.git
    ```

3. Install NPM packages
    ```sh
    cd Vacalysis-frontend && npm install
    ```
4. Run
    ```sh
    npm start
    ```
5. Open http://localhost:3000 to view it in the browser

6. For setting up backend Refer here.
